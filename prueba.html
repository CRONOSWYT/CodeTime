Sistemas principales

CodeTime está dividido en dos sistemas diferentes. Uno enfocado en documentación y aprendizaje, y otro pensado para compartir código entre usuarios.

CodeTime Wiki

La Wiki sirve para buscar información sobre código, funciones, estructuras, errores y documentación. Si encuentras una parte de código que no entiendes, la Wiki ayuda a descubrir qué hace, cómo funciona y ejemplos reales de uso.

Ejemplos Funciones • Variables • APIs • CSS • JavaScript • HTML
Documentación Explicaciones organizadas y fáciles de entender.
Aprendizaje Ideal para estudiantes y programadores.

CodeTime Preview

Preview es la parte social de CodeTime. Los usuarios pueden compartir snippets, tarjetas, funciones y proyectos completos. Los demás usuarios pueden copiar código, dar likes, guardar snippets y experimentar.
